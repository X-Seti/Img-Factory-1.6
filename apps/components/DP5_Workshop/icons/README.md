# DP5 Workshop — Custom Tool Icons

Place custom icons here named by tool_id.
Supported formats: PNG, SVG (checked in that order).

## Tool IDs
pencil, eraser, fill, spray, picker, curve, line, rect, circle,
triangle, polygon, star, select, lasso, zoom, text, crop, resize, stamp

## Size
Icons should be square. They will be scaled to the configured icon size.
Recommended: 64×64 PNG or SVG with viewBox="0 0 64 64"
